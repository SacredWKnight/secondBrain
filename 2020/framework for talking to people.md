# Framework for Talking With People
https://twitter.com/nobu_hibiki/status/1319145824018837504?s=20

- Pay attention
- Empathize
- Validate
- Communicate
- Stay on same page
- Ask questions
- Learn more
- Notice when the mind dodges questions
- Build a predictive model
- Test the model
- Ask questions governed by it.
